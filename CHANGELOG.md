# Changelog - Revordian Landing Page

Semua perubahan penting pada proyek website landing page Revordian Unbeatable SMP akan dicatat di file ini secara berkala.

## [1.0.0] - 2026-07-08
### Added
- Inisialisasi struktur proyek modern berbasis Vite, React, dan Tailwind CSS.
- Pembuatan dynamic `TopBar` dan `Footer` terintegrasi dengan GitHub Developer API.
- Desain layout `Hero` cinematic bertema fantasi medieval modern dengan visual kerajaan melayang.
- Integrasi `PremiumCards` untuk informasi IP Address, versi server, dan status realtime server.
- Animasi interaktif pada `Rules` timeline dan kartu sanksi keras.
- Grid profiling pengelola server di `Staff` terhubung langsung ke render Minecraft 3D skin CDN.
- Setup template repositori profesional (README, LICENSE, CONTRIBUTING, ISSUE_TEMPLATE, dll.).
- Optimalisasi performa Google Lighthouse, SEO meta tags lengkap, serta JSON-LD sitemap.
