# Keamanan Sistem (Security Policy)

Kami sangat berkomitmen untuk menjaga keamanan data pengguna, klan, dan performa infrastruktur server Revordian dari ancaman luar.

## Melaporkan Celah Keamanan (Vulnerabilities)

Jika Anda menemukan celah keamanan (exploit, cheat, ddos vector, bug database) di dalam website resmi maupun di dalam server Minecraft Revordian, harap laporkan langsung secara privat kepada tim pengembang kami:

1. **Jangan publikasikan celah tersebut di chat publik atau Discord.**
2. Kirim pesan pengaduan resmi secara privat kepada admin melalui kontak Developer WhatsApp atau Discord.
3. Berikan rincian instruksi reproduksi celah keamanan agar tim kami dapat melakukan perbaikan secepatnya.

Kami akan melakukan review berkala terhadap setiap laporan keamanan dan memberikan apresiasi bagi pelapor yang beriktikad baik membantu menjaga integritas server.
